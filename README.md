# Scoop-Installers
Package Management for my apps I made
Install bucket to scoop using: `scoop bucket add shadowplayzdev https://github.com/ShadowPlayzDev/Scoop-Installers`
Install a app (one from list below) using `scoop install shadowplayzdev/app` with app being app name (ex. neofetch)
